Project Overview

This project demonstrates a two-tier application deployed on AWS using best DevOps practices, including public and private subnets, load balancer, and secure networking.

Technologies Used

  • AWS (EC2, VPC, ALB, NAT Gateway)
  • Linux & Nginx
  • Git & GitHub
  • Docker / Kubernetes (Optional)

Architecture

Users access the application via an Application Load Balancer. Traffic flows to EC2 instances in private subnets while outbound internet access is handled through a NAT Gateway.
